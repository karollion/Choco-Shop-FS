# Choco-Shop FullStack App


## Description

Online store with chocolate products and other sweets.


Server run on `localhost:3030`

Data Base was in `mysql://localhost:3306/choco-shop-db`

## Installation

```bash
$ npm install 

$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
